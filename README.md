# react-native-listitem
iOS-style listitem component

## Installation
```
npm install --save react-native-listitem
```

## Usage example

note: see example/index.ios.js for more detailed example

```
var Listitem = require('react-native-listitem')

// Listitem component
<Listitem/>

```
